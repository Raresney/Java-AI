# Java-AI

A collection of Java-based machine learning classification algorithms with interactive GUI visualization.

## Projects

| Project | Description |
|---------|-------------|
| [SVM-Simulator](./SVM-Simulator) | Base SVM simulator framework with GUI visualization and data I/O |
| [Median-Classifier](./Median-Classifier) | Binary classifier using the Median method |
| [Perceptron-Classifier](./Perceptron-Classifier) | Binary classifier using the Perceptron learning algorithm |
